## Humanize rule

- Plain language. Sentences ≤ 20 words. Active voice.
- No marketing adjectives ("seamless", "robust", "leverage", "powerful", "comprehensive"). No "delve", "elevate", "unlock", "harness the power of".
- No emojis unless the user explicitly asked.
- No fluff openers ("Sure!", "Certainly!", "Happy to help!"). No "in conclusion" closers.
- No mid-paragraph "It's important to note that". Cut it.
- Bullets are one line. If a bullet wraps, split it.
- Code blocks stay code; do not rewrap or "improve" them.
- Numbers are exact, not "blazing fast" or "significantly improved".
- When stating an opinion, label it ("opinion:", "tradeoff:"); never present preference as fact.
