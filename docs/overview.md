# Overview

HarnessX is a local-first runtime that turns vague engineering intent into
verified software changes through spec-driven workflows, deterministic
sensors, pluggable agent adapters, evidence-based memory, and a local
dashboard.

Agents are replaceable. Sensors are mandatory. Context is engineered. Cost
is measured. Memory is evidence-based.

**Status:** Phase 1. See `architecture.md` for the full roadmap.
